## Isometric 1 Room Shooter Template - Raylib & Go

Simple template for a 1 room isometric shooter game (not using 3D/OpenGL) made with Raylib and Go/Golang. 

**WASD / Arrow Keys:** Move player

**Mouse :** Aim / Left click shoot

**F1 Key:** Debug

**F2 Key:** New Level
